# Amazon Affiliate Product Showcase Website

אתר אינטרנט מלא להצגת מוצרים מ-Amazon עם קישורי שותפים ויצירת סרטוני שיווק אוטומטיים.

## תכונות האתר 🌐

- ✅ **חיפוש מוצרים** - חיפוש לפי מילות מפתח, קישור מוצר, או דף קטגוריה
- ✅ **הצגת מוצרים** - כרטיסי מוצרים עם תמונות, מחירים, דירוגים
- ✅ **קישורי שותפים** - כל מוצר כולל קישור שותפים של Amazon
- ✅ **יצירת סרטונים** - יצירת סרטוני שיווק אוטומטיים לכל מוצר
- ✅ **עיצוב מודרני** - ממשק משתמש יפה וידידותי
- ✅ **תמיכה בעברית** - ממשק בעברית מלא

## התקנה והפעלה 🚀

### 1. התקן תלויות

```bash
pip install -r requirements.txt
```

### 2. הפעל את השרת

```bash
python app.py
```

השרת יעלה על `http://localhost:5000`

### 3. פתח בדפדפן

פתח את הדפדפן שלך וגש ל:
```
http://localhost:5000
```

## שימוש באתר 📱

### חיפוש מוצרים

1. **חיפוש לפי מילות מפתח** - הזן מילות מפתח (לדוגמה: "טלפון חכם")
2. **חיפוש לפי קישור** - הדבק קישור מוצר מ-Amazon
3. **חיפוש לפי קטגוריה** - הדבק קישור לדף קטגוריה

### יצירת סרטון שיווק

1. מצא מוצר ברשימה
2. לחץ על כפתור "צור סרטון"
3. הסרטון ייווצר ברקע (זה עשוי לקחת כמה דקות)
4. הסרטון יישמר בתיקייה `output_videos`

### קניית מוצר

לחץ על כפתור "קנה עכשיו" כדי לעבור לדף המוצר ב-Amazon עם קישור השותפים שלך.

## API Endpoints 🔌

### חיפוש מוצרים
```
POST /api/search
Body: { "keywords": "טלפון חכם", "store": "amazon", "count": 10 }
```

### קבלת מוצר לפי ASIN
```
GET /api/product/<ASIN>
```

### קבלת מוצר לפי URL
```
POST /api/product/url
Body: { "url": "https://amazon.com/dp/..." }
```

### יצירת סרטון
```
POST /api/video/generate
Body: { "product": {...} }
```

### בדיקת סטטוס סרטון
```
GET /api/video/status/<video_id>
```

### רשימת סרטונים
```
GET /api/videos
```

## הגדרת קישורי שותפים 🔗

ודא שיש לך:
1. חשבון Amazon Associates פעיל
2. Associate Tag מוגדר בקובץ `.env`:
   ```
   AMAZON_ASSOCIATE_TAG=your-tag-20
   ```

כל הקישורים יווצרו אוטומטית עם ה-Associate Tag שלך.
